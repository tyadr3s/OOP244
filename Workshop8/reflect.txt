In this workshop I learned about inheritance by creating multiple classes like Shape, LblShape, Line, and Rectangle. This helped me understand better how a base class can use virtual functions, and how derived classes implement their own versions of methods like draw and getSpecs. One of the main difficulties I had was fixing bugs. In this workshop I had like 10 tries to fix only one bug, especially related to extra spaces or extra new lines, so I had to carefully check the formatting many times. Another problem I had was a memory error caused by deleting the same pointer twice, which helped me understand better how dynamic memory works.
